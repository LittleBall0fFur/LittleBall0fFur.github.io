About me

My name is Naomi Poort, let me tell you a bit about myself and my interest in programming. I got my degree in Game Development last academic year and started the HBO-ICT course at NHL Stenden. My interest in programming started about four years ago when I saw code for the first time. I saw how the code was written and what the results were after executing the code. It looked like magic to me and I decided to try it out for myself. So, I started following beginner tutorials for the game engine Unity. At that moment I started to like coding more and more since it keeps challenging me every step of the way. I like to be challenged and to wrap my head around the problems I encounter during programming. This is when I decided I wanted to make programming my profesion.
